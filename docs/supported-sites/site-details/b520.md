## 备注

### Book ID 说明

`book_id` 由两部分组成: `"{catalog}_{book_id}"`。

其中:

* `book_id` 为原始书籍 ID (纯数字)
* `catalog` 为 `book_id % 1000` 的结果 (取模 1000)

!!! example "示例"
    * `123` -> `0_123`
    * `50123` -> `50_50123`

> 当仅提供纯数字时, 程序会自动补全 catalog 前缀
