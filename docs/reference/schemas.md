# Schemas

The `novelkit.schemas` module defines the structured data models used
throughout NovelKit. These include configuration objects and typed
representations of site metadata, chapters, media resources, search
results, and pipeline execution state.

These models provide consistent and well-defined structures for
communication between core components such as the fetcher, parser,
session layer, processor pipeline, and client workflow.
