# NovelKit

NovelKit 是一个异步、可扩展的小说下载与处理工具包。

支持断点续爬、多格式导出、文本处理流水线，并提供 CLI 与可选 Web 界面。

> 运行要求: **Python 3.11+** (开发环境: Python 3.13)

---

**⚠️ Project under active redevelopment.**

This repository is the successor to [`saudadez21/novel-downloader`](https://github.com/saudadez21/novel-downloader), with a partial rewrite and restructuring of core components to enable better extensibility and maintainability.
