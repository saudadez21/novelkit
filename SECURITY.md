# Security Policy

## Supported Versions

The following table lists which versions of **novelkit** currently receive security updates:

| Version                   | Supported          |
| ------------------------- | ------------------ |
| latest (PyPI release)     | :white_check_mark: |
| main (development branch) | :white_check_mark: |
| older releases            | :x:                |

Security fixes are applied to the **latest stable release** and the **main** branch only.

## Reporting a Vulnerability

If you discover a security vulnerability in **novelkit**, please report it use GitHub's "Report a vulnerability" feature on the repository page.

When reporting, please include:

1. A clear description of the vulnerability and its potential impact.
2. Steps to reproduce the issue (if applicable).
3. The affected version(s) of `novelkit`.
4. Any possible mitigation or fix suggestions.
